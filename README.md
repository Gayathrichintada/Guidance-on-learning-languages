# Guidance on Learning Languages

**Guidance on Learning Languages** is a static web project that helps users explore programming languages and web technologies. It provides structured guidance through multiple pages for different languages.  

---

## **Project Pages**

- **Home Page:** `index.html` – Overview of the platform and links to all language pages.  
- **C Language Page:** `c.html` – compiler and guidance for learning C.  
- **Java Page:** `java.html` – compiler and guidance for learning Java.  
- **Python Page:** `python.html` – compiler and guidance for learning Python.  
- **Web Technologies Page:** `web-technologies.html` – Covers HTML, CSS, JavaScript, and basic web development concepts.  
- **Styles:** `styles.css` – Common styling for all pages.  

---

## **Technologies Used**

- HTML5  
- CSS3  
- Responsive design using CSS
